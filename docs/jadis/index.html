Introduction

The Jadis crate is a Java Virtual Machine language disassembler written in Rust. This project has been written for educational purposes, however, Jadis can theoretically be used as a drop-in replacement of javap.

Using Jadis

It is fairly straightforward to use Jadis. Simply run the project and use the same command-line arguments you would use with javap. Please do note that the output format could be slightly different than the output you are used to. Having said that, though, the data in the output should be identical to the output of the disassembler that came with your JDK.
